
# 👁️ Visão Computacional

Repositório dedicado à disciplina de **Visão Computacional**, cursada durante o PPG-CC da Unifesp. Aqui serão armazenados todos os códigos, experimentos e materiais desenvolvidos ao longo dos laboratórios da disciplina.

## 📚 Sobre a disciplina

A disciplina de Visão Computacional aborda técnicas e algoritmos para extração, processamento e análise de informações a partir de imagens e vídeos. Ao longo do curso, são explorados conceitos fundamentais e aplicações práticas utilizando ferramentas modernas.

## 🎯 Objetivo do repositório

Este repositório tem como objetivo:

- Organizar os códigos desenvolvidos nos laboratórios
- Servir como material de estudo e revisão
- Documentar a evolução do aprendizado ao longo da disciplina
- Facilitar futuras consultas e reutilização de implementações

### 📌 Organização dos laboratórios

Cada pasta de laboratório contém:

- 🧠 **Códigos-fonte** desenvolvidos durante as aulas práticas
- 📄 **Materiais auxiliares** (imagens, datasets, instruções, etc.)
- 📝 Possíveis anotações ou variações implementadas

## 🚀 Como utilizar

1. Clone o repositório:

   ```bash
   git clone https://github.com/RoseJnr/VisaoComputacional.git
    ````

2. Acesse a pasta desejada:

   ```bash
   cd VisaoComputacional/LaboratorioX
   ```

3. Compile e/ou execute os códigos conforme necessário:

    ```bash
    chmod +x run.sh
    ./run.sh
    ```

## 📌 Observações

* Os códigos podem variar em organização e estilo, conforme o avanço da disciplina
* Alguns laboratórios podem depender de datasets específicos (incluídos ou não no repositório)
* Recomenda-se o uso de ambientes virtuais para gerenciamento de dependências
* Os fontes foram implementados para execução em ambiente Linux - Ubuntu 22.04 LTS;
